# Examples

These projects are small, inspectable scenarios for Fabrik behavior.
They are grouped by domain and build style where that distinction
matters.

## Rust / Granular

- `rust/granular/basic-app`: a basic TOML-declared Rust library,
  binary, and test.
- `rust/granular/build-script-cfg`: a granular Rust graph where a
  `build.rs` emits a cfg consumed by a dependent library.

## Apple / iOS

- `apple/ios/simulator-app`: a SwiftUI iOS simulator app bundle.
